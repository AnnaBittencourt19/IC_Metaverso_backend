- ##### Como o Request funciona:
	1. Tipo do request (ex: GET) / caminho (caminho tipo caminho raiz)
	2. Conjunto de cabeçalhos: metadados (informações do próprio cliente por exemplo)
	3. Dados (podemos passar dados para o servidor (como um email e senha do usuário))

O servidor vai receber a requisição e fazer algo com essa requisição

- ##### Como o Response funciona:
	1. Status code (200, o código 200 significa que deu tudo certo) código da resposta
	2. Cabeçalho: O formato da resposta (ex: JSON)
	3. Dados da resposta

Navegador (Cliente) ----> Servidor
- Servidor: Um programa em Python que fica rodando esperando que as requisições cheguem pra então enviar um resposta
