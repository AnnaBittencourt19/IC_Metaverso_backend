##### Como funciona o CURL
- Código que mostra o HTML que o chrome ia retornar (requisição)
```shell
curl localhost:8000
```

- Flags mais utilizadas no Curl: 

| Flag         | o que faz                                                                                             |
| ------------ | ----------------------------------------------------------------------------------------------------- |
| (sem nada)   | Faz um GET e mostra a saída no terminal                                                               |
| -O           | Salva a saída com o nome do servidor                                                                  |
| -I           | Mostra cabeçalhos                                                                                     |
| -X METODO    | Define o metodo HTTP ( Post, put, delete)                                                             |
| -d dados     | Envia dados em uma requisição                                                                         |
| -v (verbose) | Mostra o histórico de requisição e resposta, é o mais completo onde mostra onde pode estar o problema |
- É uma forma de falar com o servidor sem usar o navegador
