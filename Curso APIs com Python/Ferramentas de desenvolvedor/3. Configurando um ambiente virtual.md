- Cria uma cópia do interpretador Python
- Venv é o interpretador python na máquina virtual 
- É importante para deixar as dependência Python organizadas 
- Cada projeto tem uma venv
	- Cada um vai ter suas próprias dependência
	- Melhora a reprodutibilidade 
- Como criar esse ambiente virtual dos projetos e usa-lo: 
	1. Ir na pasta que está o projeto pelo terminal 
	2. Escreve isso no terminal: (para criar o ambiente)
		python3 -m venv venv
	3. Executa no terminal:  (na pasta do projeto)
		source venv/bin/activate 
		Isso ativa o ambiente virtual e usa o python nele
	4. pip install -requeriments.txt ou pip install oq_quero_instalar
	5. Vai instalar as dependências externas nesse ambiente virtual (dentro do projeto)
	6. Usar o venv como interpretador no VScode:
	view -> command palette -> Python: Select Interpreter -> Seleciona o venv
- O venv nunca deve ser enviado ao github por ser pesada e especifica para a máquina em questão, deve ficar em .gitignore
