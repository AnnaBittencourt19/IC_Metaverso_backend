- No curso FastAPI do zero! do Dunossauro ele usa pipx (que cria um ambiente virtual e isola as ferramentas nele) para instalar as ferramentas porém poderia ser usado o venv (ambiente virtual)
- Para o projeto em questão precisamos baixar as seguintes dependências: (Por enquanto)
	- Poetry
	- Poetry-plugin-shell
- Poetry: Gerencia dependências de um jeito mais rápido e fácil que o venv (o poetry unifica pip+ venv + setup.py)

**Como essas dependências seriam instaladas com venv:**
	1. Foi criado o ambiente virtual [[3. Configurando um ambiente virtual]]
	2. Essas dependências foram escritas em um documento (requirements.txt)
	3. Depois no terminal (utilizando o terminal na pasta do projeto e após seguir as configurações do ambiente virtual) foi executado o seguinte comando:
``` python
	pip install -r requirements.txt
```

**Instalando usando o pipx:**
- Instalar o pipx: 
	- No terminal:
```python
pip install --user pipx
```
- Adiciona no PATH os binários instalados pelo pipx:
```python
pipx ensurepath
```
- Instalando o Poetry: 
```python
pipx install poetry
```
- Instalando a extensão do poetry que habilita o shell: 
```python
pipx inject poetry poetry-plugin-shell
```
