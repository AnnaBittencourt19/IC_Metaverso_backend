- Vamos criar um novo projeto utilizando o Poetry
- O código será: (na pasta do projeto)
```shell
poetry new --flat fast_zero 
cd fast_zero
```
- Esse código criou a seguinte estrutura: (estrutura inicial do projeto)
![[Captura de Tela 2025-11-10 às 15.45.32.png]]
- Estando na pasta fast_zero (cd fast_zero) executar no terminal:
```
poetry env use 3.14
```
Esse comando diz que usaremos a versão 3.14 do python no projeto, e vai ser escrito isso no arquivo de configurações

**Instalando o FastAPI:** (Isso precisa ser executado no terminal dentro de fast_zero para adicionar no pyproject.toml)
```
poetry install 
poetry add 'fastapi[standard]'
```
