#### Configurando o ruff
- O ruffs é configurado em 3 tabelas distintas (a global, uma para o linter e outra para o formatador) no pypproject.toml
- A global:
```toml
[tool.ruff]
line-length = 79
extend-exclude = ['migrations']
```
- O linter:
```toml
[tool.ruff.lint]
preview = true
select = ['I', 'F', 'E', 'W', 'PL', 'PT']
```
	- O I significa Isort, que é a ordenação dos imports em ordem alfabética
	- O F significa Pyflakes, que é procurar por alguns erros em relação a boas práticas de código
	- O E significa pycodestyle que procura erros de estilo no código
	- O W siginifica pycodestyle e gera avisos de estilo de código
	- PL significa Pylint que são os "erros" em relação a boas práticas de código
	- PT significa flake8-pytest boas práticas do Pytest
- Formatador:
```toml
[tool.ruff.format] 
preview = true 
quote-style = 'single'
```
- quote-style= aspas simples

#### Configurando o pytest
```toml
[tool.pytest.ini_options]
pythonpath = "."
addopts = '-p no:warnings'
```
- Testar o pytest: (Vai fazer testes nos arquivos de teste (com test no nome))
```terminal
pytest --cov=fast_zero -vv
coverage html
```

#### Configurando o Taskipy
```toml
[tool.taskipy.tasks] 
lint = 'ruff check' 
pre_format = 'ruff check --fix' 
format = 'ruff format'
run = 'fastapi dev fast_zero/app.py'
pre_test = 'task lint' 
test = 'pytest -s -x --cov=fast_zero -vv' 
post_test = 'coverage html'
```
- Nessa configuração criamos as automatizações, ao escrever task lint por exemplo, o comando ruff check será rodado
