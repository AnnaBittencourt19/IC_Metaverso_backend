- O FastAPI é um framework baseado em funções 
- Podemos fazer um função ser acessível pela rede usando esse código:
```Python
from fastapi import FastAPI 

app = FastAPI()  

@app.get('/')  
def read_root():  
    return {'message': 'Olá Mundo!'}
```
	-  O que significa isso? @app chama o FastAPI, .get diz: isso só vai ser executado quando alguém pedir um get (pegar/ler informações). o ('/') quer dizer que é no caminho raiz www.meufastapi.com/, def read_root faz com que retorne o dicionário
