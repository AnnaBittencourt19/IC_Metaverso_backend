- O instrutor do curso indica: 
	- Taskipy: Automatizador de tarefas, padroniza o processo de desenvolvimento. Em vez de decorar comandos podemos criar "tasks" (atalhos dentro do arquivo pyproject.toml). (pyproject.toml é criado em [[3. Criando um projeto]]) (não ter que lembrar todos os comandos da aplicação)
	- Pytest: Testes automatizados, garante que a API funcione hoje e amanhã, é uma ferramenta usada no Python para testar códigos (Escrever os testes)
	- Ruff: Linter (analisador de boas práticas) + Formatador (arruma o código automaticamente), promete um código organizado, bonito e profissional  

#### Código para instala-las:
(no terminal da pasta do projeto, pasta EXATA que está o pyproject.toml)
```shell
poetry add --group dev pytest pytest-cov taskipy ruff
# group dev: grupo de dependencias 
```
